## Installation

After cloning the repository, open your terminal and run:

```
cd .\vndb-web-scraping\web-scraping\
npm install
npm start
```
once you start the api server, start the client as well

```
cd .\vndb-web-scraping\front-end\
npm install
npm run dev
```
